SMARTLIB INDEX STABLE RESTORE

Fail ini membaiki:
- login Admin/Guru
- login Ahli PSS
- OPAC awam
- loading tidak tersekat pada Google Sheet
- token frontend/backend
- ID lama vs ID baru pada login

Cara guna:
1. Dalam GitHub repo, pastikan fail ini berada di root repo dan namanya tepat: index.html
2. Jangan upload ZIP terus ke repo. Extract dahulu dan upload index.html sahaja.
3. Pastikan GitHub Pages source ialah branch main/root.
4. Selepas upload, tunggu 1-3 minit dan tekan Ctrl+F5.
5. Jika webpage tidak keluar, semak nama fail: mesti index.html huruf kecil semua.

Backend Code.gs tidak perlu ditukar jika admin pernah berjaya login sebelum ini.
